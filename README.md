# Store Management System - Inventory and Sales (Console)

A simple console-based point-of-sale (POS) and inventory management system developed in **C#**. Perfect for school projects, learning purposes, or as a foundation for a more complete system.

## Main Features

- User registration and login (Admin and Cashiers)
- Inventory management (view products, sell items)
- Sales registration with automatic total calculation
- Total earnings report
- Sales report by cashier
- Shortages tracking (products with low stock or out of stock)
- Exclusive admin menu for Admin role
- Clean and user-friendly console interface with basic input validation

## User Roles

| Role     | Main Permissions                                  |
|----------|---------------------------------------------------|
| Admin    | Everything + manage users, reset the entire system |
| Cashier  | Register sales, view inventory and earnings       |

## Requirements

- .NET SDK 6.0 or higher (recommended: .NET 8)
- Operating System: Windows, macOS, or Linux
- Terminal / Console: cmd, PowerShell, Terminal, etc. (Temporarily)
